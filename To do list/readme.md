# To-Do List Application

## Description
A modern To-Do List application to manage daily tasks.

## Features
- Add and delete tasks
- Mark tasks as completed
- Filter tasks (All / Completed / Pending)
- Local storage support
- Responsive UI

## Technologies
HTML, CSS, JavaScript
