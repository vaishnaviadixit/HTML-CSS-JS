# Catering Reservation and Ordering System


## Description
A web-based system for booking catering services with details of event, guest count, and menu.


## Features
- Add new reservations
- Displays all reservations
- Form validation
- Local storage support
- Responsive UI


## Technologies
HTML, CSS, JavaScript
